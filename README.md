## 📌 Project Overview

This repository contains **Project-Data Cleaning & Preparation**, completed as part of the **Data Analytics Internship**.

The objective of this project is to transform a raw dataset into a clean, analysis-ready dataset by applying standard data preprocessing techniques using **Python** and **Pandas**.

---

## 🎯 Objectives

- Identify missing values
- Handle null values appropriately
- Detect and remove duplicate records
- Standardize text and date formats
- Validate data integrity
- Export the cleaned dataset for further analysis

## 🔍 Data Cleaning Process

The following preprocessing steps were performed:

- Imported the dataset using Pandas
- Explored dataset dimensions and data types
- Identified missing values
- Filled missing CouponCode values
- Checked and removed duplicate records
- Verified duplicate Order IDs
- Removed unnecessary whitespace from text columns
- Standardized text formatting
- Validated date formats
- Verified numerical columns
- Exported the cleaned dataset
